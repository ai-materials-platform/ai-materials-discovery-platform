# AI 소재 발굴 플랫폼 (AI Materials Discovery Platform) v2.0

본 플랫폼은 스테인리스강 소재의 화학 조성과 공정 변수를 기반으로 주요 기계적 물성을 예측하고 분석하는 인공지능 시스템입니다. 이전 버전에서 확장되어 강도뿐만 아니라 연성 지표까지 포함한 종합적인 분석을 제공합니다.

## 🚀 주요 신규 기능 (v2.0 업데이트)

### 1. 4대 핵심 물성 통합 예측
기존의 강도 지표에 재료의 변형 능력을 나타내는 연성 지표를 추가하여 총 4가지 물성을 동시에 예측합니다.
*   **강도(Strength):** 0.2% 항복강도 (Yield Stress), 인장강도 (UTS)
*   **연성(Ductility):** 연신율 (Elongation), 단면 수축률 (Area Reduction)

### 2. 지능형 학습 제어 시스템
*   **사용자 정의 반복 횟수(Iterations):** 데이터의 특성에 따라 학습의 깊이를 직접 조절할 수 있습니다 (기본 2,000회).
*   **조기 종료 (Early Stopping):** 학습 중 더 이상 성능 향상이 없으면 AI가 스스로 판단하여 학습을 중단함으로써 과적합(Overfitting)을 방지하고 모델을 보호합니다.

### 3. 고도화된 데이터 시각화
*   **이중 Y축 예측 그래프:** 단위가 다른 강도(MPa)와 연성(%)을 한 그래프에서 명확하게 비교할 수 있는 이중 축 시스템을 도입했습니다.
*   **상세 성능 분석 (Parity Plot):** 실제값(Actual)과 예측값(Predicted)의 분포를 대각선(y=x) 지표와 함께 제공하여 모델의 정확도를 직관적으로 검증합니다.
*   **반응형 상관관계 히트맵:** 창 크기에 따라 수치 표시 여부가 자동으로 조절되어, 작은 화면에서는 깔끔하게, 큰 화면에서는 상세하게 데이터를 분석할 수 있습니다.

## 🛠 시스템 아키텍처

### Backend Engine
*   **DataEngine:** `STMECH_AUS_SS.xls` 데이터를 로드하고 4개 타겟 변수에 대한 전처리를 수행합니다.
*   **ModelEngine:** Random Forest, GBM, MLP, TFP(확률론적 앙상블) 등 4가지 모델을 지원하며 다중 출력(Multi-output) 예측에 최적화되어 있습니다.

### Frontend GUI
*   **PyQt6 기반 인터페이스:** 모든 메뉴와 안내 문구를 한국어로 최적화하였으며, '맑은 고딕' 글꼴을 적용하여 가독성을 높였습니다.
*   **실시간 학습 피드백:** 학습 진행 상태 및 신뢰도(R²) 점수를 실시간으로 제공합니다.

## 📋 사용 방법
1.  **데이터 선택:** 학습용 엑셀 파일을 선택합니다.
2.  **모델 및 설정:** 원하는 AI 모델과 최대 반복 횟수를 설정하고 '학습 시작'을 누릅니다.
3.  **성과 확인:** '상세 성능 분석' 탭에서 모델이 실제 데이터를 얼마나 잘 따라가는지 확인합니다.
4.  **물성 예측:** 화학 조성과 공정 조건을 입력하여 4가지 핵심 물성을 즉시 예측합니다.
